# Plan: Improve UI + gate commits on `impeccable detect`

## Goal
1. Block git commits whenever `impeccable detect` reports findings in shipped source.
2. Fix all real impeccable anti-patterns so the gate passes.

## Findings triage
- `vendor/**` (Laravel mail, phpunit, symfony CSS) — third-party → exclude from scan.
- `scratch_stitch/**` (Stitch design mockups, not shipped) — throwaway → exclude from scan.
- `resources/css/app.css` + `resources/js/pages/*.tsx` — real → fix.

`impeccable detect` exits `2` on findings, `0` when clean → drives the hook.

## Steps

### 1. Commit gate
- `.githooks/pre-commit`: run `bunx impeccable detect resources/ --quiet`, abort on non-zero exit with a clear message (+ `--no-verify` escape hatch).
- `git config core.hooksPath .githooks` (shareable, tracked).
- `package.json` script `"impeccable": "impeccable detect resources/"`.
- `.impeccable/config.json`: `detector.ignoreFiles` for `vendor/**` and `scratch_stitch/**`.

### 2. app.css
- `.gradient-text-primary` / `.ai-gradient-text` → solid `#004ac6` (drop background-clip:text).
- `.ai-gradient-bg` → blue gradient `#004ac6 → #2563eb`.
- Body/label font vars: Inter → Hanken Grotesk; keep Manrope for display.

### 3. Palette: purple/violet/indigo/#7C3AED → brand blue + neutrals
Files: admin-dashboard, assistant, dashboard, explorer-dashboard, workspace, welcome.
- purple-*/indigo-*/#7C3AED → blue scale / text-primary.
- Gradients recolored to blue/slate (no indigo/cyan).

### 4. Remove side-tab thick borders
- Active nav: drop `border-r-4 border-primary`, keep `bg-primary/10 text-primary font-bold`.
- Chat bubbles: drop `border-l-4`, keep bg.
- Alert cards: `border-l-4 border-red-500` → `border border-red-200 bg-red-50/60` + icon chip.

### 5. Bounce easing
- workspace.tsx typing dots: `animate-bounce` → staggered `animate-pulse`.

### 6. Font heads
- app.blade.php + welcome.tsx Google Fonts: drop Inter, add Hanken Grotesk (keep Manrope + Material Symbols).

### 7. Verify
- `bunx impeccable detect resources/` → 0 findings.
- `npm run types:check`, `npm run lint:check`.
- Test hook blocks dirty / passes clean.

## Decisions
- Scan scoped to `resources/`; vendor + scratch_stitch excluded.
- Accent = brand blue `#004ac6`; fonts = Manrope (display) + Hanken Grotesk (body).
